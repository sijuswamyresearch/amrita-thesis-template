# images/

Place all your image files here.

Supported formats: PDF (for vector graphics), PNG, JPEG.

## Required files
- `amrita-logo.png`  — University logo (download from https://amrita.edu)

## Naming convention
Use descriptive lowercase names with hyphens:
  ch1-system-overview.pdf
  ch3-proposed-architecture.png
  ch4-result-comparison.pdf

## Tips
- Use PDF/EPS for figures containing text or line art (stays sharp at any size).
- Use PNG at ≥ 300 dpi for photographs or screenshots.
- Keep file sizes reasonable — large images slow down compilation.
